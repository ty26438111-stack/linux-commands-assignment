# 📌 Linux Commands: top, ps, jobs, kill

이 문서는 top, ps, jobs, kill 명령어에 대해 정리한 것입니다.

---

## 🖥️ 1. top
### ✔️ top 설명
`top` 명령어는 시스템 프로세스와 CPU/메모리 사용량을 실시간으로 모니터링하는 명령어이다.

### ✔️ 주요 옵션
| 옵션 | 설명 |
|------|------|
| `-b` | 배치 모드 출력 |
| `-n` | 반복 횟수 지정 |
| `-u` | 특정 사용자 프로세스만 표시 |

### ✔️ 실행 예시
```bash
top
```

---

## 🖥️ 2. ps
### ✔️ ps 설명
`ps`는 현재 실행 중인 프로세스 정보를 출력하는 명령어이다.

### ✔️ 주요 옵션
| 옵션 | 설명 |
|------|------|
| `-e` | 모든 프로세스 출력 |
| `-f` | full 포맷 출력 |
| `aux` | BSD 스타일 전체 프로세스 출력 |

### ✔️ 실행 예시
```bash
ps aux
```

---

## 🖥️ 3. jobs
### ✔️ jobs 설명
`jobs`는 현재 터미널에서 실행 중이거나 백그라운드에서 실행 중인 작업 목록을 보여준다.

### ✔️ 실행 예시
```bash
jobs
```

---

## 🖥️ 4. kill
### ✔️ kill 설명
`kill` 명령어는 프로세스에 종료 신호를 보내는 명령어이다.

### ✔️ 주요 옵션
| 옵션 | 설명 |
|------|------|
| `-9` | 강제 종료(SIGKILL) |
| `-15` | 정상 종료(SIGTERM) |

### ✔️ 실행 예시
```bash
kill -9 1234
```
